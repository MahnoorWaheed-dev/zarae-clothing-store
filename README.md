# ✨ ZARAE – Timeless Elegance

**ZARAE** is a modern women's fashion website inspired by the grace and richness of Pakistani couture. Designed with a focus on elegance, smooth user experience, and refined aesthetics, this project reflects a digital space where tradition meets contemporary style.

---

### 🌐 Live Demo  
[Click Here to Visit 🚀](https://mahnoorwaheed-dev.github.io/zarae-clothing-store/)

---

## 🌙 About The Project

ZARAE is not just a website — it’s a visual experience. From **Timeless Elegance** to **Pakistani Couture**, every section is crafted to highlight the beauty of eastern fashion in a modern interface.

The design focuses on clarity, soft transitions, and a luxurious feel, ensuring users can explore collections effortlessly across all devices.

---

## 🖤 Key Features

* 🌗 Light & Dark Mode for a seamless viewing experience
* 📱 Fully Responsive Design (Mobile, Tablet & Desktop)
* 🎞️ Smooth transitions and interactive UI
* 🛍️ Product-focused layout with clean structure
* 🎨 Elegant color palette inspired by fashion brands
